# SQL-queries
The cleaning product database was created taking into account its category, segment, total sales, calendar, and region. In addition, searches were conducted for product groups with specific characteristics.
Specific SQL searches were performed.
• The best-selling products are liquid bleach in the Mexico region, with 195,877 units sold for a total value of 5,521,416.
• The least-selling products are towel-based in the other segment, with 204 units sold for a value of 15,041.
• The next least-selling products are liquid sanitizers, with 714 units sold for a value of 81,915.
